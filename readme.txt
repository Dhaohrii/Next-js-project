We Review

We Review is a dynamic web application designed to allow users to comment, like, and review their favorite coffee shops, restaurants, and bars. This platform provides a space for food and drink enthusiasts to share their experiences and discover new places based on community feedback. The project leverages modern web technologies to deliver a responsive and engaging user experience.

Project Structure

1. Core Features
Commenting and Reviewing: Users can write detailed reviews about their experiences at various coffee shops, restaurants, and bars.
Liking System: Each review can be liked by other users, highlighting the most popular opinions.
User Profiles: Users have personal profiles where they can view and manage their reviews.
Search and Discovery: Users can search for specific places or browse by category to discover new locations.
2. Administrative Tools
Content Moderation: Admins can review and moderate user-generated content to maintain quality and compliance with community guidelines.
Analytics Dashboard: Provides insights into user engagement, popular locations, and overall platform performance.
Technologies Used

Frontend: Next.js, Tailwind CSS
Backend: Node.js, Express.js
Database: MySQL
Design: Figma
Version Control: Git
